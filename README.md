## Penjelasan Tugas
Project ini merupakan simulasi gerbang logika digital menggunakan Tinkercad.

Pada rangkaian ini terdapat beberapa jenis gerbang logika, yaitu:
- AND
- OR
- NOT
- NAND
- NOR
- XOR
- XNOR

Setiap gerbang logika diuji menggunakan input switch dan output LED untuk melihat hasil logika dari masing-masing rangkaian.

Tujuan dari project ini adalah untuk memahami cara kerja dasar gerbang logika dalam sistem digital dan elektronika.
